# Loan Approval Prediction Machine Learning Project

## Overview
This project aims to develop a machine learning model that predicts the likelihood of loan approval based on various input features provided by the applicant. The model is trained on a dataset that includes historical loan data, applicant information, and whether the loan was approved or not.

## Dataset
The dataset used for this project is derived from public sources and includes the following features:
- Applicant's Gender
- Marital Status
- Number of Dependents
- Education Level
- Employment Status
- Applicant's Income
- Loan Amount
- Credit History
- Property Area

## Methodology
1. **Data Preprocessing:** 
   - Cleaning the data by handling missing values and outliers.
   - Encoding categorical variables using techniques such as one-hot encoding.
   - Normalizing numerical features to improve model performance.

2. **Feature Selection:** 
   - Using feature importance techniques to select the most impactful features for the model.

3. **Modeling:** 
   - Implementing different algorithms including Logistic Regression, Decision Trees, and Random Forest.
   - Tuning hyperparameters to optimize model performance.

4. **Evaluation:** 
   - Using metrics like accuracy, precision, recall, and F1-score to evaluate model performance on the test dataset.
   - Employing confusion matrix and ROC curve for further analysis.
